# 🎵 אפליקציית כתיבת שירים עם אקורדים

אפליקציה מלאה לכתיבת, עריכה וניהול שירים עם אקורדים בפורמט ChordPro.

---

## 🚀 הפעלה (פשוט!)

### **לחץ פעמיים על:**
```
RUN.bat
```

זהו! האפליקציה תיפתח.

### **לעצירה:**
```
STOP.bat
```

או סגור את שני החלונות (Backend + Frontend)

---

## 📋 דרישות

- ✅ **Python 3.11+** - [הורד](https://www.python.org/downloads/)
- ✅ **Node.js 18+** - [הורד](https://nodejs.org/)

---

## 🎸 שימוש

### 1️⃣ יצירת תיקייה
לחץ "תיקייה חדשה"

### 2️⃣ יצירת שיר
היכנס לתיקייה → "שיר חדש"

### 3️⃣ כתיבת אקורדים
```
[C]על כל [Am]אלה, [F]על כל [G]אלה
[C]שמור נא [Am]לי א-[F]לי הטוב [G]שלי
```

**חשוב:** אקורדים בסוגריים `[C]` לפני ההברה
